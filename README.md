# Municipal Link

## À propos du projet

Ce projet est un système de gestion du patrimoine municipal, conçu pour faciliter la gestion des biens, des locations, des ventes et des paiements.

## Technologies utilisées

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- PHP (Backend)
- MySQL (Base de données)

## Installation Locale

### Frontend

1. Clonez le dépôt.
2. Installez les dépendances : `npm install`
3. Lancez le serveur de développement : `npm run dev`

### Backend

1. Installez XAMPP.
2. Déplacez le contenu du dossier `backend` dans `C:\xampp\htdocs\municipal-link-api` (ou un dossier de votre choix).
3. Importez la base de données en utilisant le script `database.sql`.
4. Configurez la connexion dans `backend/config/db.php`.
